# Quran Reader — পারা ১

প্রতিদিন ৫ মিনিট কুরআন পড়ুন।

## Features
- Arabic + বাংলা + English translation
- 15 parts (~10 ayahs each, ~5 min reading)
- Progress tracking (saved in browser localStorage)
- PWA — install on phone homescreen, works offline
- Mobile-first responsive design

## Deploy to Vercel
1. Push to GitHub
2. Go to vercel.com → Import → Select repo
3. Click Deploy (auto-detected as Vite)

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
